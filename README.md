b. Menggambar 3 titik
c. Menggambar garis
d. Menggambar bangun primitif
e. Menambahkan animasi
f. Menambahkan interaksi pengguna
g. Menggambar bangun 3 dimensi
h. Pencahayaan dan pembayangan
i. Material
